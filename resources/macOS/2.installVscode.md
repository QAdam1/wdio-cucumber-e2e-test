## Install VSCode

### Pre-check
1. Open `terminal` and check if code is installed already
    - `code -v` 
2. Ignore if already installed

### Steps
1. Download VScode
    - [Download VSCode](https://code.visualstudio.com/download)


### Post-check
1. Check if node is sucessfully installed
    - `code -v` -> `x.xx.x`
